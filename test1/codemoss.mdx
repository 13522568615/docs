## CodeMoss的安装与配置详解

为了帮助大家更好地使用CodeMoss，以下将详细介绍插件的安装与配置步骤。

### 步骤一：在VSCode中安装CodeMoss插件

1. **打开VSCode编辑器**：
   首先，确保您已经安装了最新版本的Visual Studio Code。打开VSCode，进入主界面。

2. **搜索CodeMoss**：
   在搜索框中输入“CodeMoss”，直接安装`ChatGPT - ChatMoss（CodeMoss）`或者`ChatGPT - 中文版`，这两个工具是同一家产品，所以安装任意一个就可以。![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/4becc9315a3b4e4f99e0055d7c60c73d.png)

### 步骤二：配置服务地址

1. **打开设置**：
   安装完成后，点击VSCode右下角的“设置”按钮，或通过菜单栏选择`File -> Preferences -> Settings`进入设置页面。

2. **查找CodeMoss设置**：
   在设置搜索框中输入`CodeMoss`，找到`Code Moss: Server Url`选项。

3. **设置服务器URL**：
   在`Server Url`字段中输入相应的服务器地址，如图所示：![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/b2eb79b4132e4fbfa0853f19e86c9200.png)
